## 🎯 Features

- Fully functional **5-page** interactive website prototype
- Homepage with hero, featured recipes & categories
- Recipe listing with real-time search, filters & sorting
- Detailed recipe view with ingredients checklist and step-by-step instructions
- User profile page with My Recipes & Favorites
- Upload recipe form with dynamic fields and image upload
- Smooth navigation, hover effects, and responsive design

All built with clean frontend code by a **Frontend Website Developer**.

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Tailwind via CDN)
- Vanilla JavaScript
- Font Awesome icons

**Single-file prototype** — perfect demonstration of frontend development skills.
